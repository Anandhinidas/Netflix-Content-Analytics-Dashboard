# 🎬 Netflix Content Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Netflix content and provides insights into movies and TV shows available on the platform.

The dashboard helps users understand:

- Total content available on Netflix
- Distribution of Movies vs TV Shows
- Top countries producing Netflix content
- Most popular genres
- Content growth over time
- Rating distribution
- Monthly content additions
- Average movie duration trends

---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## Dataset Information

Dataset contains:

- Show ID
- Title
- Type
- Country
- Genre
- Rating
- Release Year
- Date Added
- Duration

Total Records: **8,808**

---

## Key Insights

### Content Distribution
- Movies: 6,131
- TV Shows: 2,676

### Global Reach
- Content available across 123 countries.

### Popular Genres
- International Movies
- Drama
- Comedy
- Documentaries

### Content Growth
- Netflix content increased rapidly after 2015.
- Peak additions occurred between 2018 and 2020.

### Ratings Analysis
- TV-MA and TV-14 dominate the platform.

## Data Preparation

Performed data cleaning using Power Query:

- Fixed date format issues
- Created Year column
- Created Month Added column
- Split Country field
- Split Genre field
- Removed errors and null values
- Created lookup tables for analysis
